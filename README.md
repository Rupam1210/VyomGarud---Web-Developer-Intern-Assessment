# VyomGarud---Web-Developer-Intern-Assessment
VyomGarud – UAV Landing Page 🚁⚡

A modern, military-grade landing page built for VyomGarud, showcasing precision engineering, autonomy, and high-reliability drone technology.
Developed with React + TypeScript and styled using Tailwind CSS for clean, responsive, and futuristic UI.

⚙️ Tech Stack

React (TypeScript)

Tailwind CSS

Framer Motion (subtle animations)

Poppins / Inter / Montserrat fonts

🧩 Page Structure
Hero Section

Bold title, tagline, CTA button, and a dominant UAV/drone visual.

About

Short mission statement describing VyomGarud’s purpose and engineering ethos.

Capabilities

3–4 cards highlighting UAV features, systems, or offerings.

Highlights

Quick bullets emphasizing reliability, autonomy, and engineering precision.

Contact / Footer

Simple contact form or link section.

Install dependencies:

npm install


Run development server:

npm run dev


Build for production:

npm run build
